## 📌 重點說明

**關於「25歲以下」的觀察：**
YC W26 批次中的AI應用公司創辦人，大多具有以下背景：
- 擁有PhD學位（物理、CS、工程等）
- 曾在SpaceX、Nvidia、Tesla、McKinsey等頂級公司工作多年
- 具備深厚的產業經驗

這顯示目前**AI應用層的創業門檻較高**，需要結合領域知識+技術能力，因此25歲以下的創辦人較少見於這個批次。

## 📊 趨勢分析

### 應用層AI的五大方向（2026年2月）

| 方向 | 代表公司 | 說明 |
|------|----------|------|
| **1. 內容創作自動化** | Wideframe | 影片編輯工作流程 |
| **2. 金融科技AI** | Proximitty | 貸款服務與催收 |
| **3. 農業AI** | OctaPulse | 水產養殖自動化 |
| **4. 國防安全AI** | DroneTector | 無人機偵測 |
| **5. 企業流程AI** | Reframe, Cozmo | 採購、營運自動化 |

### 共同特點
1. **垂直領域深耕**：不只做通用AI，而是深入特定產業（金融、農業、國防、太空）
2. **Agentic AI**：從「工具」進化為「AI員工」，能執行完整工作流程
3. **整合既有軟體**：與Adobe Premiere、銀行系統等現有工具整合，而非取代
4. **收入導向**：多家公司已有6位數以上年合約或千萬級收入

## 💡 給尋找「25歲以下AI創業家」的建議

如果目標是找**25歲以下**的AI應用創業家，建議關注：

1. **更早期的孵化器/加速器**
   - 高中生/大學生創業計畫（如Thiel Fellowship、各種hackathon）
   - 大學創業社團（如Stanford的StartX學生團隊）

2. **消費級AI應用**
   - AI交友、AI遊戲、AI內容創作等年輕人更熟悉的領域
   - TikTok/Instagram上的AI工具創作者

3. **開源社群**
   - GitHub上的熱門AI專案年輕貢獻者
   - Hugging Face上的年輕開發者

4. **即將到來的YC批次**
   - Summer 2026（2026年6月）可能會有更多學生創業團隊
