# 美國矽谷AI年輕創業家資訊整理

## 1. Anthropic — Dario Amodei & Daniela Amodei

### 創辦人背景
- **Dario Amodei** (CEO) 和 **Daniela Amodei** (總裁) 是兄妹關係
- 兩人皆為前 OpenAI 核心成員，Dario 曾任 OpenAI 研究副總裁
- 2021年與其他5位前 OpenAI 員工共同創立 Anthropic

### 創業內容
- **公司定位**：AI 安全研究公司，以「公共利益公司」(Public Benefit Corporation) 形式運作
- **核心產品**：Claude 系列大型語言模型（Claude 4 Opus/Sonnet 等）
- **技術特色**：Constitutional AI（憲法式AI），讓AI遵循一套人類價值原則

### 創業過程與里程碑
| 時間 | 事件 |
|------|------|
| 2021年 | 公司成立，獲得5.8億美元融資（含FTX的5億） |
| 2023年3月 | 推出 Claude 1 & Claude Instant |
| 2023年9月 | Amazon 投資12.5億美元（後增至80億） |
| 2023年10月 | Google 投資5億美元（後增至20億） |
| 2024年 | 估值達183億美元（F輪融資130億） |
| 2025年2月 | G輪融資300億美元，估值達**3,800億美元** |
| 2026年2月 | 在超級盃投放廣告，成為OpenAI主要競爭對手 |

### 特色
- 專注於 AI 安全與對齊（Alignment）研究
- 推出 Model Context Protocol (MCP) 成為產業標準
- 與美國國防部合作，同時堅持不干預政策限制

## 2. Cohere — Aidan Gomez

### 創辦人背景
- **Aidan Gomez** (CEO)，英裔加拿大人
- **年齡**：約28歲（1990年代末出生）
- **學歷**：多倫多大學資工與數學學士，牛津大學博士（2024年取得）
- **關鍵經歷**：2017年時年僅20歲，在Google Brain實習期間共同撰寫了改變AI界的論文《Attention Is All You Need》（Transformer架構）

### 創業內容
- **公司定位**：企業級生成式AI平台
- **核心產品**：
  - Command A 系列語言模型
  - Cohere North（安全AI工作平台）
  - Aya Vision（多語言開源模型）
- **目標市場**：金融、醫療、製造、能源等受監管行業

### 創業過程與里程碑
| 時間 | 事件 |
|------|------|
| 2019年 | 與 Nick Frosst、Ivan Zhang 共同創立 Cohere |
| 2021年9月 | A輪融資4,000萬美元（Index Ventures領投） |
| 2022年2月 | B輪融資1.25億美元（Tiger Global領投） |
| 2023年6月 | C輪融資2.7億美元，估值22億美元 |
| 2024年7月 | 融資5億美元，估值55億美元 |
| 2025年8月 | 融資5億美元，估值**70億美元** |
| 2025年 | 聘請前Meta AI負責人 Joëlle Pineau 擔任首席AI官 |

### 特色
- **Transformer 共同發明人**：Gomez是當年8位作者之一
- 與Oracle、SAP、Dell等企業巨頭深度合作
- 獲加拿大政府2.4億美元投資建設主權AI數據中心
- 入選Time 100 AI 2023最具影響力人物

## 3. Scale AI — Alexandr Wang

### 創辦人背景
- **Alexandr Wang** (創辦人)
- **年齡**：28歲（1997年出生）
- **學歷**：MIT輟學生（17歲入學，後休學創業）
- **經歷**：曾在Quora與Lucy Guo共事

### 創業內容
- **公司定位**：AI數據標註與模型評估平台
- **核心業務**：
  - 數據標註（Scale AI）
  - 眾包平台（Remotasks）
  - LLM評估與對齊（SEAL Lab）
- **客戶**：OpenAI、Google、Microsoft、Meta、美國國防部

### 創業過程與里程碑
| 時間 | 事件 |
|------|------|
| 2016年 | 與Lucy Guo共同創立，通過Y Combinator孵化 |
| 2018年 | Lucy Guo離開公司 |
| 2019年8月 | Peter Thiel的Founders Fund投資1億美元，成為獨角獸（估值10億+） |
| 2021年 | 估值達73億美元 |
| 2023年8月 | 成為OpenAI GPT-3.5微調「首選合作夥伴」 |
| 2024年5月 | 估值達140億美元（Amazon、Meta投資） |
| 2025年1月 | 與美國AI安全研究所合作 |
| 2025年6月 | **Meta以148億美元收購49%股權**，Alexandr Wang加入Meta |

### 特色
- **全球最年輕的獨角獸創辦人之一**：22歲時公司估值破10億
- 軍事合作：與美國國防部簽訂多項AI合約（包括Thunderforge項目）
- 發布「Humanity's Last Exam」AI基準測試

## 4. Perplexity AI — Aravind Srinivas

### 創辦人背景
- **Aravind Srinivas** (CEO)
- **學歷**：印度理工學院畢業，後赴UC Berkeley攻讀博士
- **經歷**：曾在OpenAI、Google Brain、DeepMind擔任研究員

### 創業內容
- **公司定位**：AI原生搜尋引擎
- **核心產品**：
  - Perplexity（免費版）
  - Perplexity Pro（付費版，支援多種LLM後端）
  - Perplexity Assistant（AI助理）
  - Perplexity Comet（AI瀏覽器）
- **技術特色**：結合LLM與即時網路搜尋，提供帶來源的答案

### 創業過程與里程碑
| 時間 | 事件 |
|------|------|
| 2022年8月 | 公司成立 |
| 2022年12月 | 推出搜尋引擎產品 |
| 2023年4月 | 獲得2,600萬美元融資，推出iOS App |
| 2024年4月 | 估值達10億美元（獨角獸） |
| 2025年1月 | 提議與TikTok US合併（被拒） |
| 2025年6月 | 估值達140億美元 |
| 2025年9月 | 估值達**200億美元** |
| 2025年8月 | 出價345億美元收購Google Chrome |

### 特色
- 挑戰Google搜尋霸主地位
- 投資人包括Jeff Bezos、Nvidia、C羅（Cristiano Ronaldo）
- 與多家媒體（NYT、BBC等）有版權爭議
- 2025年月均查詢量達7.8億次

## 5. Character.ai — Noam Shazeer & Daniel de Freitas

### 創辦人背景
- **Noam Shazeer** (CEO) 和 **Daniel de Freitas**
- **經歷**：兩人皆為Google資深工程師
- **關鍵貢獻**：
  - Shazeer是《Attention Is All You Need》論文的主要作者之一
  - de Freitas是Google LaMDA（對話AI）的主要設計師

### 創業內容
- **公司定位**：AI角色對話平台
- **核心產品**：Character.ai（c.ai）
- **功能**：用戶可創建、訓練並與各種虛擬角色對話（名人、虛構人物、原創角色）

### 創業過程與里程碑
| 時間 | 事件 |
|------|------|
| 2021年11月 | 公司成立，獲得4,300萬美元種子輪 |
| 2022年9月 | Beta版上線，首週吸引數十萬用戶 |
| 2023年3月 | A輪融資1.5億美元，估值達**10億美元**（獨角獸） |
| 2023年5月 | 推出iOS/Android App，首週下載量超170萬 |
| 2024年8月 | **Google聘請Noam Shazeer回鍋**，簽訂技術授權協議 |
| 2024年12月 | 因青少年用戶安全問題推出新防護措施 |
| 2025年10月 | 宣布禁止18歲以下用戶創建或對話 |

### 特色
- 主要用戶群為16-30歲年輕人
- 2024年1月日活躍用戶達350萬
- 面臨多起訴訟（涉及青少年心理健康、內容審核等）

## 總結比較

| 公司 | 創辦人 | 年齡 | 估值（2025-2026） | 核心領域 |
|------|--------|------|-------------------|----------|
| Anthropic | Dario & Daniela Amodei | 30-40歲 | 3,800億美元 | 安全對齊AI |
| Cohere | Aidan Gomez | ~28歲 | 70億美元 | 企業級LLM |

## 共同趨勢

1. **Google Brain/OpenAI 出身**：多數創辦人來自這兩個AI研究重鎮
2. **Transformer 論文作者**：Cohere的Gomez和Character.ai的Shazeer都是《Attention Is All You Need》作者
3. **快速獨角獸化**：這些公司多在3-5年內達到10億+估值
4. **與大廠博弈**：既與Google、Microsoft、Amazon合作，又與其競爭
5. **AI安全與監管**：隨著規模擴大，都面臨安全、版權、隱私等挑戰
